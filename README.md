ในเว็บไซต์มีข้อมูล

-Average_Windspeed/Max_Windspeed

-Average_Temperature/Max_Temperature/Min_Temperature

-Average_Humidity/Max_Humidity/Min_Humidity

-Average_Pressure/Max_Pressure/Min_Pressure

-Precipitation

-Average_Visibility/Max_Visibility/Min_Visibility
