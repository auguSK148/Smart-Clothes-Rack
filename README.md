# Smart-Clothes-Rack
AI builder project

สาเหตุที่ผมเลือกทำโครงงานนี้เพราะว่า เนื่องจากผมเคยทำโครงงาน ราวตากผ้าอัจฉริยะ มาก่อน ทำให้ผมเริ่มคิดว่าเราจะสามารถพัฒนาระบบให้ฉลาดขึ้นได้อย่างไร
จากปัญหานี้ ผมจึงอยากลองพัฒนา AI ที่สามารถช่วยทำนายว่าฝนอาจจะตกในช่วงประมาณ 30–60 นาทีข้างหน้า โดยเน้นการคาดการณ์เฉพาะพื้นที่บ้านหรือบริเวณใกล้เคียง เพราะแอปพยากรณ์อากาศทั่วไปมักจะทำนายในระดับทั้งเมือง แต่ในความเป็นจริงบางครั้งฝนอาจตกเฉพาะบางพื้นที่เท่านั้น
แนวคิดของโครงงานคือการนำข้อมูลสภาพอากาศ เช่น ความชื้น อุณหภูมิ และความกดอากาศ จากปีก่อนๆแล้วนำมาวิเคราะห์ว่ามีความน่าจะเป็นที่ฝนจะตกเท่าไร
